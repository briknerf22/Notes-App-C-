# Moje Poznámky App 📝
Webová aplikace pro správu poznámek vytvořená v ASP.NET MVC Core.

## Funkce
- Registrace a přihlášení (BCrypt hashování hesel)
- Vytváření, mazání a řazení poznámek
- Označování poznámek jako "Důležité" a filtrování
- Kompletní zrušení účtu

## Technologií
- **Backend:** ASP.NET Core 9.0
- **Databáze:** PostgreSQL (Supabase)
- **Deployment:** Render.com
